# Corpse-System

This is a San Andreas Multiplayer corpse system demo for *[hopefully modular] samp/open.mp gamemodes* by **spicybung**. Since I was using it in my server, I decided to try and touch it up, then post it here, as I'm not working on my gamemode at the current time. Granted, it's not perfect, but it should do the trick.
